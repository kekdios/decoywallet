# Release APKs

This directory contains release builds of the Bitcoin Vault Android app.

## Files

- `decoy_wallet-release-1.0.apk` - Release build of the app (signed with debug keystore for testing)

## Installation

1. Transfer the APK to your Android device
2. Enable "Install from Unknown Sources" in Settings
3. Open the APK file and install

## Note

These APKs are signed with the debug keystore for testing purposes. For production distribution, a proper release keystore should be used.
